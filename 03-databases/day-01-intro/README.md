# Day 1: Introduction to Databases

Welcome to Day 1 of your journey into the world of data! Today is all about understanding the fundamental structure of where information lives.

## Core Concepts
Before writing code, you need to understand the "Map":

1. **Database**: A container for all your project's data.
2. **Table**: A structured collection of data (like a spreadsheet tab).
3. **Column**: A specific attribute (e.g., "Username", "Email"). Called a *field*.
4. **Row**: A single record in a table. Called a *tuple*.

## Why PostgreSQL?
PostgreSQL is an **Object-Relational Database Management System (ORDBMS)**. It's famous for:
- Being open-source and free.
- Extreme reliability (ACID compliance).
- Handling complex data types beyond just text and numbers.
