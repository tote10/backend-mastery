# Day 2: Basic SQL – SELECT & WHERE

Now that you have a table, you need to know how to talk to it. Today is about **Queries**—asking the database for specific information.

## Learning Objectives
1. **SELECT**: The fundamental way to fetch data.
2. **Columns**: Choosing specific fields vs. using `*`.
3. **WHERE**: Filtering your results so you only get what you need.
4. **Operators**: Using `=`, `<>`, `>`, `<`, `BETWEEN`, `LIKE`, and `IN`.

## Key Syntax
```sql
SELECT column1, column2 
FROM table_name 
WHERE conditions;
```
